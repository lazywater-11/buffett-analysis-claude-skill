# 巴菲特/段永平 价值投资分析框架

## 核心哲学

> 「我们是商业分析师，不是股票分析师、市场分析师，也不是宏观经济分析师。」——巴菲特
>
> 「看不懂的不做。投资就是要找到一门好生意，然后等待好价格。」——段永平

**串行关卡制**：6 关必须按顺序通过。任何一关 FAIL，立即输出 AVOID 结论并停止分析，不浪费时间做估值。大多数公司应该在前 3 关被淘汰。

**偏向淘汰**：不确定时选择不做，这本身就是巴菲特/段永平的核心原则。

## 关卡 1：真的看得懂这门生意吗？（Business Comprehension）

**核心问题：我对这门生意有没有真正的理解？**

巴菲特：「如果我不能用一段话解释这门生意，我不会投。」
段永平：「看不懂的不做。这不是认输，这是边界感。」

**必须能清晰回答以下三个问题——模糊=看不懂：**

**问题 1：这家公司靠什么赚钱？为什么客户付钱给它而不是竞争对手？**
（不是描述产品，而是说清楚：利润从哪里来？定价权从哪里来？）

**问题 2：为什么这家公司 10 年后还会存在，而且可能更强？**
（不是说"市场需求大"，而是：什么结构性因素让它的地位难以被颠覆？）

**问题 3：如果今天买入并锁仓 10 年不能卖，我有没有足够的信心？**
（去掉短期股价波动，只看生意本身，你敢持有吗？）

**三种「假懂」陷阱——触发即降为 ⚠️ 部分看懂：**
- 陷阱 1：「它是行业龙头所以好」— 龙头地位≠护城河，为什么是龙头？护城河是什么？
- 陷阱 2：「市场很大，增速很快」— 市场大≠公司好，谁能拿到这个市场？
- 陷阱 3：「所有人都在用它」— 普遍使用≠有定价权，用户是因为依赖还是因为没有选择？

**三级评定：**

✅ **看懂**：三个问题都能清晰作答，且未触发假懂陷阱
⚠️ **部分看懂**：能回答部分问题，或触发了一个假懂陷阱（继续分析，但最终结论保守）
❌ **看不懂**：核心盈利机制说不清楚，或触发了两个及以上假懂陷阱

**输出格式：**
```
### 关卡 1：能否看懂这门生意 ✅ 看懂 / ⚠️ 部分看懂 / ❌ 看不懂

**问题 1（靠什么赚钱）：** [回答]
**问题 2（10年后为何更强）：** [回答]
**问题 3（锁仓10年的信心）：** [回答]
**假懂陷阱检查：** 无 / [触发的陷阱]

**判断：** 看懂 / 部分看懂 / 看不懂
**理由：** [简要说明]
```

> ❌ 看不懂 → 输出：「此公司目前不在可分析范围内，建议持续观察而非买入。」停止分析。

## 关卡 2：商业模式质量（Business Model Quality）

**核心问题：这是一门好生意吗？**

**「消失测试」**（段永平最常用的思维工具）：
> 如果这家公司明天消失，它的客户会不会很难受？有多难受？

**评估项（基于 STEP 1 数据）：**

1. **复购性**：产品/服务是客户反复购买的，还是一次性的？
   - 高复购：软件订阅、白酒、必需消费品 → 有利
   - 低复购：房产、大型设备 → 需要其他优势补偿

2. **资本密度**：轻资产还是重资产？
   - 轻资产（软件、品牌消费品）→ 更优
   - 重资产（矿产、制造业）→ 可接受，但 ROE 要高

3. **自由现金流**：FCF 是否持续为正？（直接读取脚本输出的 FCF 数据）

4. **段永平「本分测试」**：
   - 公司有没有明确的「不做什么」边界？
   - 管理层是在做对用户好的事，还是只追短期利润？
   - 有没有跟风追热点（元宇宙、区块链、AI 蹭热度）的行为记录？

5. **毛利率**：是否稳定或上升？（直接读取脚本输出的毛利率趋势）

**FAIL 触发条件：**
- 连续 3 年以上 FCF 为负（且不是成长期投入）
- 毛利率持续下滑超过 5 个百分点
- 过去 5 年内有明显「什么热做什么」的战略漂移记录
- 消失测试：客户完全不会难受（完全可替代的同质化产品）

**输出格式：**
```
### 关卡 2：商业模式 ✅ PASS / ⚠️ WARN / ❌ FAIL

**消失测试：** [如果这家公司消失，客户反应是？]

**复购性：** 高/中/低
**资本密度：** 轻/中/重
**FCF 状况：** [读取脚本数据，列近3年FCF趋势]
**毛利率趋势：** [读取脚本数据]
**本分测试：** PASS/WARN/FAIL

**判断：** PASS/WARN/FAIL
**理由：** [关键点说明]
```

> ❌ FAIL → 输出最终判断「🔴 回避」，原因：商业模式不符合好生意标准。
> ⚠️ WARN → 继续分析，但在最终判断中降低评级。

## 关卡 3：护城河宽度（Competitive Moat）

**核心问题：什么阻止了有钱的竞争对手抢走市场份额？**

巴菲特：「我们的核心竞争力是护城河，而非城堡。」
段永平：「好公司的用户是因为习惯和依赖，而不仅仅是因为便宜。」

**护城河类型识别（选最符合的 1-2 种）：**

| 类型 | 特征 | 典型案例 |
|------|------|---------|
| 品牌溢价 | 同类产品客户愿意多付钱 | 茅台、苹果、可口可乐 |
| 转换成本 | 离开代价很高 | 企业软件、银行账户、微信 |
| 网络效应 | 用户越多越有价值 | Visa、微信、美团 |
| 成本优势 | 规模/资源/流程带来的低成本 | 紫金矿业（矿产资源）、宁德时代 |
| 监管/专利壁垒 | 牌照、专利保护 | 医院、券商、部分药企 |

**护城河耐久性评估：**
- 10 年内，这条护城河会变宽还是变窄？
- AI/技术革命能打穿这个护城河吗？
- 段永平关键问：客户是因为「离不开」还是因为「最便宜」而留下？

**ROE 作为护城河代理指标**（读取脚本数据）：
- ROE 持续 >20%：强护城河的财务证明
- ROE 持续 15-20%：中等护城河
- ROE < 15% 且下滑：护城河可能在弱化

**FAIL 触发条件：**
- 唯一竞争优势是价格最低（无定价权）
- 市场份额连续 3 年下滑且无反转迹象
- 行业格局已出现明显颠覆者（Uber 之于出租车）
- 护城河在 10 年内大概率消失

**输出格式：**
```
### 关卡 3：护城河 ✅ PASS / ⚠️ WARN / ❌ FAIL

**护城河类型：** [类型]
**宽度评估：** 宽/中/窄
**耐久性：** 10 年内变宽/稳定/变窄
**ROE 印证：** [读取脚本数据，近5年ROE趋势]
**最大威胁：** [描述]

**判断：** PASS/WARN/FAIL
**理由：** [关键点]
```

> ❌ FAIL → 输出最终判断「🔴 回避」，原因：缺乏可持续护城河。

## 关卡 4：管理层可信度（Management Integrity）

**核心问题：这些人值得信任吗？他们像股东一样思考吗？**

段永平：「管理层不好的公司，再便宜我也不碰。这是我唯一的红线。」
巴菲特：「我们寻找的管理者，是把公司当作自己唯一资产来经营的人。」

**评估项（结合脚本数据 + WebSearch）：**

1. **股权结构**：读取脚本输出的前十大股东数据，确认：
   - 创始人/核心管理层持股比例（>10% 为佳）
   - 大股东是否在减持？

2. **资本分配历史**（最重要的检验）：
   - 分红历史：读取脚本分红数据，是否稳定分红/增长？
   - 并购历史：是否有高溢价烂并购？（WebSearch 补充）
   - 融资历史：是否频繁稀释股东？

3. **诚实度测试**：
   - 公司出问题时，管理层是主动披露还是掩盖？
   - 业绩不好时是找借口还是承担责任？

4. **段永平「本分」检验**：
   - 管理层是否有明确的长期战略而非追季度 KPI？
   - 有无道德问题记录？

**一票否决红旗（任意一条 → 立即 FAIL）：**
- 财务造假历史（无论多久前）
- 大规模关联交易/明显利益输送
- 频繁高溢价收购后商誉减值
- 管理层薪酬与公司业绩严重背离

**输出格式：**
```
### 关卡 4：管理层 ✅ PASS / ⚠️ WARN / ❌ FAIL

**持股情况：** [来自脚本前十大股东数据]
**资本分配评分：** 优/良/差（分红/并购/融资历史）
**诚实度测试：** PASS/WARN/FAIL
**本分检验：** PASS/WARN/FAIL
**一票否决项：** 无/[具体问题]

**判断：** PASS/WARN/FAIL
**理由：** [关键点]
```

> ❌ FAIL → 输出最终判断「🔴 回避」，原因：管理层可信度存疑。这是不可妥协的底线。

## 关卡 5：10 年后的世界（Future Durability）

**核心问题：10 年后，这门生意比今天更好还是更差？**

巴菲特测试：「如果我把这家公司锁起来 10 年不看，我还放心吗？」
段永平偏好：「10 年后基本面不会变的生意——好的消费品牌、优质金融机构。」

**评估项：**

1. **行业大趋势**：
   - 顺风行业（人口老龄化、能源转型、AI 基础设施）
   - 逆风行业（传统媒体、燃油车、线下零售）
   - 周期性行业（大宗商品、地产）→ 需特别注意估值时机

2. **技术颠覆风险**：
   - AI 会替代这家公司的核心价值吗？
   - 有无清晰的技术颠覆者已经在路上？

3. **监管风险**：
   - 政策依赖度高吗？（教培、医疗、互联网平台曾因此受创）
   - 反垄断/数据安全风险？

4. **人口/地理结构**：
   - 目标客户群体在 10 年内增长还是萎缩？

**FAIL 触发条件：**
- 核心产品面临清晰且快速的替代威胁（已有成熟替代品）
- 严重依赖特定政策，且政策方向已在转向
- 目标市场 10 年内结构性萎缩

**输出格式：**
```
### 关卡 5：10 年耐久性 ✅ PASS / ⚠️ WARN / ❌ FAIL

**行业趋势：** 顺风/中性/逆风/周期性
**技术颠覆风险：** 低/中/高
**监管风险：** 低/中/高
**10 年后判断：** 更好/持平/更差

**判断：** PASS/WARN/FAIL
**理由：** [关键点]
```

> ❌ FAIL → 输出最终判断「🔴 回避」，原因：商业模式耐久性不足。

## 关卡 6：估值与建仓（Valuation & Entry Point）

**只有前 5 关全部 PASS（或 PASS+WARN）才到这里。**

段永平：「好公司 + 合理价格 = 买入。不要等完美价格错过好公司，但也不要以任何价格买入。」
巴菲特：「宁愿以合理价格买入优秀公司，也不愿以优惠价格买入平庸公司。」

### A. 自由现金流收益率（FCF Yield）
**直接读取脚本输出的 FCF Yield 数据：**
```
FCF Yield = 年自由现金流（最新年度）÷ 当前总市值

参考线：
> 8%：明显低估，强烈吸引
5%-8%：合理，可考虑建仓
3%-5%：略贵，等待或轻仓
< 3%：较贵，需要更高增长预期支撑
```

### B. 归一化 PE
**直接读取脚本输出的归一化PE数据（3-5年均值净利润计算）：**
```
参考线（段永平观点）：
优质稳定增长公司：20-25x PE 是合理价格
高成长公司：可适当放宽至 30-35x
周期性公司：PE 意义有限，看 PB 和资产价值
```

### C. 内在价值估算（私人买家视角）
```
简化 DCF：
内在价值 ≈ 年 FCF × (1 ÷ 折现率)
折现率通常用 8-10%（等效 10-12.5x FCF）

加：成长溢价（如果公司还在高速增长）
减：风险折扣（负债、政策风险等）
```

### D. 安全边际与建仓建议

```
当前价格 vs 内在价值估算：

内在价值明显高于市价（>30% 折扣）：
→ 🟢 建议建仓，当前价格有较好安全边际

内在价值与市价接近（±15% 以内）：
→ 🟡 合理价格，可分批建仓或等待更好价格

市价高于内在价值（>15% 溢价）：
→ 🟡 等待，好公司但当前价格不划算
```

**输出格式：**
```
### 关卡 6：估值 🟢 建仓 / 🟡 等待 / 🔴 太贵

**FCF Yield：** [X]%（来自脚本数据）
**归一化 PE：** [X]x（来自脚本：[N]年均值净利润）
**PE(TTM)：** [X]x（来自脚本：5年分位 [X]%）
**内在价值估算：** [X] 元/股（说明方法和假设）
**当前价格：** [X] 元（来自脚本）
**安全边际：** [溢价 X% / 折价 X%]

**建仓建议：** 立即建仓 / 分批建仓 / 等待回调至 [X] 元 / 观望
```

## 重要注意事项

1. **周期性行业特殊处理**（矿业、钢铁、地产）：
   - 关卡 5 自动降为 WARN（周期性是内在属性）
   - 关卡 6 估值改用 PB、资产重置价值而非 PE/FCF
   - 关卡 6 需额外判断当前处于周期的什么位置（底部/中段/顶部）
   - 最终结论需注明「周期顶/底部」的判断

2. **中国公司特殊风险**：
   - 关卡 4 管理层需额外关注：大股东减持（对比脚本股东数据前后期变化）
   - 关卡 5 需额外考虑：政策监管风险、VIE 结构风险（如适用）

3. **数据不足时的处理**：
   - 脚本无法获取数据时（港股/美股某些字段），用 WebSearch 补充
   - 不要用「大概」「可能」来填补数据空白，宁可说「数据不足，无法判断关卡 X」

4. **这不是 stock-analysis skill 的替代**：
   - 本 skill 判断「值不值得投」（商业质量 + 估值逻辑）
   - stock-analysis skill 判断「什么时候买」（技术面 + 短期趋势）
   - 两者配合使用效果最佳
